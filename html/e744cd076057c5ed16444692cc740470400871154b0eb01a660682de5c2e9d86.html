0.2 Stochastic Process

A stochastic process is a collection of random variables XtX_tXt​ indexed by time ttt.

0.3 Markov Property

A stochastic process {Xt}\{X_t\}{Xt​} has the Markov property if for any n≥0n \geq 0n≥0, any t0<t1<⋯<tnt_0 < t_1 < \cdots < t_nt0​<t1​<⋯<tn​, and any B∈BB \in \mathcal{B}B∈B, we have

0.4 Markov Chain

A Markov chain is a stochastic process with the Markov property.

0.4.1 Transition Probability

The transition probability at time nnn of a Markov chain is defined as

Pij=P(Xn+1=j∣Xn=i).P_{ij} = P(X_{n+1} = j | X_n = i).Pij​=P(Xn+1​=j∣Xn​=i).

The transition probability matrix at time nnn of a Markov chain is a matrix P=(Pij)P = (P_{ij})P=(Pij​) where PijP_{ij}Pij​ is the transition probability from state iii to state jjj.

0.4.3 n-Step Transition Probability

The n-step transition probability from time mmm to n+mn+mn+m of a Markov chain is defined as

0.4.4 Time-Homogeneous Markov Chain

A Markov chain is time-homogeneous if the transition probabilities are independent of time.

In this note, we focus on time-homogeneous Markov chains.

0.7 Communication

We say that the state iii communicates with state jjj if

Pij(n)>0 for some n≥0P_{ij}^{(n)} > 0 \text{ for some } n \geq 0Pij(n)​>0 for some n≥0

and is denoted by i→ji \rightarrow ji→j.

0.7.1 Inter-Communication

Two states iii and jjj are said to inter-communicate if i→ji \rightarrow ji→j and j→ij \rightarrow ij→i.

0.7.2 Irreducible Class of States

A set of states CCC is said to be an irreducible class of states if any state in CCC communicates with any other state in CCC.

0.7.3 Closed Class of States

A set of states CCC is said to be a closed class of states if any state in CCC communicates only with other states in CCC.

0.7.4 Absorbing State

A state iii is said to be an absorbing state if itself forms a closed class of states.

0.8 Recurrence

0.8.1 Recurrent and Transient State

A state iii is said to be recurrent if the hitting probability

A class of states CCC is said to be recurrent if any state in CCC is recurrent.

A class of states CCC is said to be transient if any state in CCC is transient.
