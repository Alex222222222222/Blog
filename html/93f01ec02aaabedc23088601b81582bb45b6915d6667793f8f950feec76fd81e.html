0.2 Introduction

For long, MacOS users have been left out of the gaming world. But with the advent of Wine, it's now possible to run Windows games on MacOS. In this tutorial, I'll show you how to run Witcher 3 on MacOS using Whisky - a modern Wine wrapper built for MacOS.

All software used in this tutorial is free and open-source except for Steam and Witcher 3, which are proprietary.

0.3 Prerequisites

0.3.1 Steam Account

You'll need to have a Steam account and have already purchased Witcher 3.

0.3.2 System Requirements

You'will have to have Rosetta 2 installed on your system.

From Apple's Official Documentation:

Open any app that needs Rosetta. If the app opens, Rosetta is already installed and working. If Rosetta is not installed, youâ€™ll be automatically asked to install it.

I am using MacOS Sonoma 14.2 on a MacBook Pro (M1, 2021). I do not ensure that this method will work on other systems.

0.3.3 Download Necessary Files

0.3.3.1 Download Whisky

Download the latest version of Whisky from the official GitHub Releases Page. Just use the latest version available.

Download the .zip file or .dmg file as I highlighted in the image below.

0.3.3.2 Download Steam Installer for Windows

Download the Steam installer for Windows from the official Steam Website.

Important: What you need is the Windows installer, not the MacOS installer. The file should be named something like SteamSetup.exe.

To download it, you'll need to visit the Official Steam Website and click on the Windows Icon below the "INSTALL STEAM" button as I highlighted in the image below.

0.4 Installation

0.4.1 Install Whisky

Open the .zip or .dmg file you downloaded on Download Whisky step and drag the Whisky app to your Applications folder. Or just leave it anywhere on your computer if you prefer.

0.4.2 Install Necessary Dependencies for Whisky

Open Whisky app, and it will ask you to install GPTK. Click on "Next" and wait for the installation to finish.

0.4.3 Create a New Bottle

Create a new bottle by clicking on the "+" button on the right top corner of the Whisky app.

Just put arbitrary bottle name and leave all other settings as default and click "Create". You will need to wait some time for the bottle to be created.

0.4.4 Install Steam

On the right bottom corner of the Whisky app, click on the "Run..." button.

Select the Steam installer you downloaded on Download Steam Installer for Windows step. And then click "Open".

You will see the Steam installer running. Just click "Next" and "Install" as you would do on Windows.

Just wait a bit, and your Steam will be installed and opened.

0.4.5 Install Witcher 3

Now, you can install Witcher 3 as you would do on Windows. Just search for Witcher 3 in your library and click on "Install".

0.5 Final Thoughts

This method also allows you to run other Windows games on MacOS. However, not all games are supported, and some games may not run as smoothly as they would on Windows.

For some common issues and solutions, you can check the Whisky's Official Documentation.
